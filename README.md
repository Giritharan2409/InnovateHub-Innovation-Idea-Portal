# 💡 InnovateHub: Innovation Idea Portal

InnovateHub is a ServiceNow application that provides a centralized platform for employees to submit, review, approve, and track innovation ideas within an organization. The application replaces manual idea collection methods with an automated workflow, improving collaboration, transparency, and efficiency.

---

## 📌 Project Overview

InnovateHub streamlines the complete innovation lifecycle by allowing:

- Employees or Students to submit innovative ideas.
- Reviewers to evaluate submitted ideas.
- Managers to approve or reject reviewed ideas.
- Users to track the status of their ideas in real time.

The application is built completely on the ServiceNow platform using native features such as custom tables, forms, ACLs, Client Scripts, UI Policies, Flow Designer, Reports, and Dashboards.

---

## 🎯 Objectives

- Centralize innovation idea management.
- Reduce manual approval processes.
- Improve collaboration among employees.
- Secure data using Role-Based Access Control (RBAC).
- Automate review and approval workflows.
- Provide visibility through reports and dashboards.

## 🔐 Security

The application uses Role-Based Access Control (RBAC).

| Role | Access |
|------|--------|
| User | Submit ideas and view own records |
| Reviewer | Review ideas and update review details |
| Manager | Approve or reject reviewed ideas |

ACLs ensure users only access permitted records and fields.
